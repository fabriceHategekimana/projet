Interface
==========

Définit quel méthodes une classe devrait impémenter.
Permet d'implémenter des mêmes comportements pour les mêmes classes.
C'est ce qui compose le polymorphisme quand une ou plusieurs classes utilisent la même interface.

Une interface est une classe abstraite qui:
- N'a pas de propriété.
- qui ne peut pas avoir de methode protected
- Tout les méthodes sont abstraites par défaut
- S'implément au lieu de s'hériter
