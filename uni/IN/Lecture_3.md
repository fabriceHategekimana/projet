# Image acquisition and sensing

Pipeline of image processing
Modèles mathématiques
Image= vecteur
On parle d'imperfection

soit une image f donnée

g= PHf+zo

H: "imaging" operator (bluring operator= kill high frequency)
P: sampling
z: noise (=bruit)

Quantization: mappage d'un ensemble continue à un ensemble discret

Sampling: élément prit selon le delta t. extraction simple. Problème: les élément extrait appartiennent à l'ensemble continue donc l'ensemble d'arrivé est R.

Quantization: Les éléments sont forcé selon un ensemble d'arrivé. Avantage: compression optimisé. L'image perdcomplètement de son naturel selon l'ensemble d'arrivé.

Chaque appareil a une contrainte H.
