Wrapper (Adapter)
=================

Permet de changer un contract dans un autre contract. Évite le cas par cas.

On enveloppe en fait une class [nom] dans une wrapper qu'on appelle souvent [nom]Wrapper.

Le wrapper permet de "coller au mieux" le contrat d'un code avec un autre.

Si une class a un "final" définit, alors la classe ne peut pas servir d'extention.
