# Donnez la définition d’un PGM en relation avec les notions de probabilités. Quelle est l’utilité d’utiliser un PGM ?
PGM (probabilistic graphical model) est un graphe décrivant les relation de dépendances/indépendances probabiliste entre variables aléatoires.

# Donnez un exemple d’utilisation. Quel est l’impact de l’indépendance conditionnelle dans les PGM ?
Très souvent utilisé pour prendre des décisions selon plusieurs paramètres (dont on a plusieurs échantillons) donnés.
Si les variables aléatoires (réprésentant les données) sont toutes indépendantes les une des autres alors on est obligé de toutes les prendre en compte pour donner certaines réponses, ce qui peut augmenter le temps de calcule de la solution.
