Sources:
=========

C'est le moyen d'être indépendant et de pouvoir se développer à son rythme.

Les différentes sources:
- help
- plugins
- communauté


## Help:
La première resource disponible est la commande :help disponible au sain de vim.

Commandes utiles:
:help		-> recherche simple
:helpgrep	-> recherche de termes dans la documentation

zone utiles
:help help-summary	-> expliques les différentes commande de help
:help options.txt	-> indique les différentes options qu'on peut "set" dans vim.
:help eval.txt		-> regroupe ce qui concèrne le langage Vimscript
